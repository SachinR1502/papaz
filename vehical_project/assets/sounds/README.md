Please add the following MP3 files to this directory (`assets/sounds/`):

1. **broadcast.mp3** - Played when a new broadcast job is available (e.g., a "ping" or "radar" sound)
2. **assignment.mp3** - Played when a job is directly assigned (e.g., a "chime" or "bell")
3. **update.mp3** - Played for job updates (e.g., a "pop")
4. **payment.mp3** - Played when payment is received (e.g., "cha-ching")
5. **message.mp3** - Played for chat messages or orders (e.g., "click")
6. **notification.mp3** - General fallback sound

## Where to find sounds?
You can download free notification sounds from:
- [Mixkit](https://mixkit.co/free-sound-effects/notification/)
- [Pixabay](https://pixabay.com/sound-effects/search/notification/)

## Configuration
Mappings are defined in `services/soundNotificationService.ts`.
